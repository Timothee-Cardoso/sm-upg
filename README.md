# Usage
<!-- usage -->
```sh-session
$ npm install -g @secman/sm-upg
$ sm-upg COMMAND
running command...
$ sm-upg (-v|--version|version)
@secman/sm-upg/0.1.1 linux-x64 node-v15.10.0
$ sm-upg --help [COMMAND]
USAGE
  $ sm-upg COMMAND
...
```

## `sm-upg help [COMMAND]`

```

  $ sm-upg help [COMMAND]

  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
