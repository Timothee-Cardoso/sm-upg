# Install

```bash
npm install -g @secman/sm-upg
```

# Usage
```sh-session
$ sm-upg COMMAND

running command...
$ sm-upg (-v|--version|version)

@secman/sm-upg/0.1.3 linux-x64 node-v15.10.0
$ sm-upg --help [COMMAND]
USAGE
  $ sm-upg COMMAND
```

# Commands
* [`sm-upg help [COMMAND]`](#sm-upg-help-command)
* [`sm-upg start`](#sm-upg-start)

## `sm-upg help [COMMAND]`

display help for sm-upg

```
USAGE
  $ sm-upg help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

## `sm-upg start`

Start Upgrade secman

```
USAGE
  $ sm-upg start
```
